Compositor Random Arduino

Este proyecto consiste en la creación de música aleatoria mediante la implementación de reglas musicales en un dispositivo Arduino Uno. El algoritmo de generación de notas, programado en lenguaje C++, se ejecuta en el dispositivo y utiliza la señal de ruido eléctrico ambiente capturado por una entrada analógica del Arduino, la cual es digitalizada por su convertidor analógico a digital de 10 bits, como fuente de aleatoriedad. Estas lecturas se utilizan para determinar la frecuencia de las notas generadas, con mayor o menor probabilidad según las expectativas teóricas de la música, y para establecer el volumen de cada nota. La duración de las notas se determina mediante un valor de tiempo generado aleatoriamente y se envía un mensaje asincrónico (noteOff) para apagar la nota después de que se haya completado el tiempo. Las notas generadas se envían a través de un puerto serial a un software de producción musical, que se encarga de reproducir el sample correspondiente a cada nota, junto con el instrumento especificado por el algoritmo de generación.
